# brian.vn site

Static website for `brian.vn`, deployed with GitHub Pages.

## Files

- `index.html`: landing page
- `privacy.html`: privacy policy
- `terms.html`: terms of use
- `app-ads.txt`: AdMob verification file
- `CNAME`: custom domain for GitHub Pages
- `.nojekyll`: serve static files directly

## Before publishing ads

Replace `pub-REPLACE_WITH_YOUR_PUBLISHER_ID` in `app-ads.txt` with your real AdMob publisher ID.
